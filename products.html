Not sure which board fits your project? The ESP32 is the safe default for anything needing Wi-Fi and Bluetooth together, while the NodeMCU is a lighter option if you only need Wi-Fi.

Arduino Uno is the classic beginner board: It uses an 8-bit processor, has no built-in wireless features, and excels at simple, repetitive tasks like reading a sensor or blinking LEDs.
